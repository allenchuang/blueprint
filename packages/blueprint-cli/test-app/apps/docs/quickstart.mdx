## Prerequisites

- Node.js 18+
- pnpm 10+
- Neon PostgreSQL database (or local PostgreSQL)

## Setup

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone <repo-url>
    cd mastermind
    pnpm install
    ```
  </Step>
  <Step title="Configure environment">
    Copy the example env files and fill in your values:
    ```bash
    cp .env.example .env
    ```
    Set your `DATABASE_URL` to your Neon connection string.
  </Step>
  <Step title="Push database schema">
    ```bash
    pnpm db:push
    ```
  </Step>
  <Step title="Start development">
    ```bash
    pnpm dev
    ```
    This starts all apps in development mode.
  </Step>
</Steps>

## Running Individual Apps

```bash
pnpm dev:web      # Web app on port 3000
pnpm dev:admin    # Admin panel on port 3002
pnpm dev:server   # API server on port 3001
pnpm dev:docs     # Documentation
```
