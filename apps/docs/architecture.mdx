## Project Structure

```
mastermind/
├── apps/
│   ├── web/            # Main web application
│   ├── admin/          # Admin panel
│   ├── docs/           # Documentation (Mintlify)
│   ├── react-native/   # Mobile app (Expo)
│   ├── remotion/       # Video generation
│   └── server/         # API server (Fastify)
├── packages/
│   ├── db/             # Database schema & client
│   ├── eslint-config/  # Shared ESLint configuration
│   └── typescript-config/ # Shared TypeScript configuration
└── turbo.json
```

## Design Decisions

- **Independent shadcn installs**: Each Next.js app has its own shadcn/ui components, allowing different themes per app.
- **Shared database package**: `@repo/db` provides a single source of truth for the database schema, shared by web, admin, and server.
- **Expo Router**: The React Native app uses file-based routing for consistency with Next.js.
- **Fastify + Swagger**: API server auto-generates OpenAPI documentation.
