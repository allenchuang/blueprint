## What is Blueprint?

Blueprint is a **full-stack monorepo starter kit** built with [Turborepo](https://turbo.build/). It gives you a complete, production-ready codebase the moment you run `npx blueprint new` — including a web app, admin panel, mobile app, API server, documentation site, and video generation tool, all wired together and ready to ship.

The goal is simple: **eliminate the setup tax**. Authentication, payments, database, API, internationalization, voice agents — all built-in, all optional, all gracefully removable via the CLI.

> **"Ship ideas at the speed of thought."**

## Monorepo layout

```
blueprint/
├── apps/
│   ├── web/          → Next.js 15 + shadcn/ui + Tailwind CSS v4   :3000
│   ├── admin/        → Next.js 15 + shadcn/ui + Tailwind CSS v4   :3002
│   ├── docs/         → Mintlify documentation                      :3003
│   ├── react-native/ → Expo + Expo Router + NativeWind
│   ├── remotion/     → Video generation                            :3004
│   └── server/       → Fastify + Swagger API                       :3001
├── packages/
│   ├── app-config/   → Single source of truth for branding & config
│   ├── blueprint-cli/→ npx blueprint CLI
│   ├── db/           → Drizzle ORM schema + Neon client
│   ├── eslint-config/→ Shared ESLint rules
│   └── typescript-config/ → Shared tsconfig
└── prompts/          → AI prompt library
```
