## Overview

Blueprint is a full-stack application built with a modern monorepo architecture.

### Tech Stack

| App | Stack |
|-----|-------|
| Web | Next.js, shadcn/ui, Tailwind CSS |
| Admin | Next.js, shadcn/ui, Tailwind CSS |
| Mobile | Expo, Expo Router, NativeWind |
| Server | Fastify, Swagger |
| Database | Drizzle ORM, Neon PostgreSQL |
